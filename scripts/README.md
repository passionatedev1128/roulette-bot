# Utility Scripts

This directory contains utility and diagnostic scripts for the Roulette Bot project.

## Script Categories

### Diagnostic Scripts
- `diagnose_*.py` - Diagnostic tools for debugging detection, frame capture, mouse movement, etc.

### Template Creation
- `capture_*.py` - Scripts for capturing templates and coordinates
- `create_*.py` - Scripts for creating templates from video or grid
- `template_*.py` - Template-related utilities

### Testing & Analysis
- `analyze_*.py` - Analysis scripts for test results
- `verify_*.py` - Verification scripts
- `check_*.py` - Setup and configuration checks

### Development Tools
- `coordinate_*.py` - Coordinate capture and management tools
- `visual_*.py` - Visual debugging helpers
- `fix_*.py` - Fix scripts for specific issues
- `run_*.py` - Runner scripts for specific components

## Usage

Most scripts are for development and debugging purposes. Refer to individual script docstrings for usage instructions.

## Note

These scripts are not required for normal bot operation. They are provided for development, testing, and troubleshooting.
